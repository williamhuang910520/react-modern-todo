# React-Modern-TODO
這是一個幫助你規劃生活大小事的 TODO kanban

## 特色
#### ✏️ 基本筆記編輯
將滑鼠移到筆記下方，可進行編輯刪除操作。移到筆記右上方可釘選。
#### 📄 新增筆記
將滑鼠移到欄位最下方以新增筆記。
#### 🔍 搜尋筆記
利用上方的搜尋欄位以及Tag選擇，可以快速尋找筆記。
#### 🤚 拖曳筆記
可以將筆記拖曳到其他欄位。

## 使用技術
React + Tailwind + ReactDND
